
# JSP/Servlet을 이용한 컴퓨터판매 컨셉 웹사이트 구축


### 주요기능
1. 회원가입
2. 로그인
3. 관리자(회원정보/문의글 확인)  
4. 카테고리(상품 검색)
5. 사용자 견적 추천(구현시 구현자의 인공지능 기술에 제한이 있어 더미형태로 만듬)
6. 커뮤니티(CRUD 갤러리형식 게시판)
